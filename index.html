Loan Prediction Analysis

- Analysed and sampled HMDA dataset of over 15 million rows and 99 columns using various data wrangling techniques
- Performed LASSO Regression to filter irrelevant features and accounted for class imbalance using SMOTE
- Tuned and tested classifiers such as Random Forest, SVM and AdaBoost to predict whether a customer is approved to take a loan and evaluated model performance using Precision-Recall Curve and ROC curves
